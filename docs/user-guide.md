# 使用说明

## 教学目标

本系统用于桥梁结构认知、无人机巡检航线规划和裂缝智能检测教学。经纬度、病害与 AI 结果均为教学模拟数据，不能直接替代实际桥梁安全鉴定。

## 操作流程

1. 点击“生成桥梁场景”，等待 6 跨预应力混凝土简支高架桥完成装配。
2. 在三维区按住鼠标左键旋转，使用滚轮缩放，按住鼠标右键平移；“重置视角”可返回全桥视图。
3. 点击“生成巡检航线”，观察桥面俯视、桥梁两侧以及桥底绕墩四类航线。航点间隔约 2 m，点击任一航点可查看教学模拟经纬度、相对高度、速度、云台俯仰角、偏航角及拍摄任务。
4. 点击“AI 裂缝检测”，观察图像载入、预处理、模型推理、像素分割、尺寸测量和三维匹配过程。
5. 检测完成后，从右侧列表选择裂缝。三维相机会移动到对应病害，裂缝旁直接展开带分割标注的拍摄图像、构件位置、最大宽度和置信度；右侧详情区仍可切换原始图像和分割结果。

## 辅助功能

- “航线”控制航线与航点的显示状态。
- “环绕”开启或暂停自动旋转。
- 用户开始旋转、平移或缩放时，自动环绕与自动飞行会立即停止，视角保持在用户操作后的画面。
- “全屏”进入沉浸式教学演示。
- “说明”展示实训流程与注意事项。
